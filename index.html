Progressive Web Apps

Progressive Web Apps are web apps that use web technologies to bring a native app-like user experience to cross-platform web applications. They bridges the gaps between responsive web pages and mobile apps.

Progressive Web Apps are light weight, fast and highly scalable. With a Progressive Web App you can build one application that will run on any operating system.
